# Usage

See all `:playerProps` at [`asciinema player` docs](https://github.com/asciinema/asciinema-player)

```md

    <Asciinema src="/casts/demo.cast" :playerProps="{speed: 2, rows: 15}"/>
</RenderWhen>
```
